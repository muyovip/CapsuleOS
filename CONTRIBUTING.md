## The Ritual of Contribution

1. **Fork the Temple**: Create your own liminal space.  
2. **Clone the Sanctum**: `git clone your-fork`.  
3. **Branch the Path**: `git checkout -b revelation/your-feature`.  
4. **Code with Reverence**: Follow deterministic patterns. Use tokenized tensors for non-arbitrary inference.  
5. **Test as Meditation**: Run 311 tests. `cargo test`.  
6. **Commit as Offering**: `git commit -m "Revelation: description — cos(Blue) = 1"`.  
7. **Push the Energy**: `git push origin revelation/your-feature`.  
8. **Pull Request as Petition**: Open PR with cosmic description. Reference Work Orders.  
9. **Merge as Ascension**: Await the High Priest's blessing.
